Usage

Upload a json file with the fields "value" , "year" and "country code" where country code is the three letter ISO 3166 code or use one of the example files below.

  • The choropleth map will be colored according to the current year's data.
  • The play button on the bottom animates the change of the value over the years
  • With the next,previous and reset buttons the user can navigate through the years
  • The continent button colors the countries according to the continent's (unweighted) average
  • The summary box provides a quick view over the most important statistics of the dataset
  • Clicking any country brings up a line graph of the current dataset for the selected country along with the continent's average

The user can upload multiple datasets. Each of them has its own color which is used as base color for the choropleth map but also the line graphs When more than one dataset is loaded, clicking any country will bring up line charts of all the datasets in a single view
